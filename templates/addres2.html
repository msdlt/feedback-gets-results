{{survey}}

Resultsets are linked to feedback Surveys, so that feedback can be monitored in conjuction. Courses can be run multiple times in a year, therefore there is a Survey Instance for each time the Survey is used, as a Survey collects feedback for each Course.

Survey Instances are used on the Medical Sciences Division Feedback System, which allows Feedback gets Results (this system) to monitor which students are providing feedback for which Resultset. Results can either be withheld until feedback is provided, or simply used to check who has given feedback for a Resultset.
