Resultsets are stored in an Excel spreadsheet with three levels of headers, followed by an infinite number of students (example spreadsheet)

Headings can have any number of subheadings, but subheadings can have only one heading. Each field heading can have one subheading, or one heading; a subheading is assigned to the most recent heading used, but subheadings, like headings, are optional. Field headings are required as they specify what each value represents.

First time?

You will require the master Excel spreadsheet, laid out for use with Feedback gets Results. If you are unsure of how to use the spreadsheet, please refer to the explanation spreadsheet and the example spreadsheet for further guidance.
